MarkDown入门教程#

开始#

一、准备工作#

  1. 安装VScode

  2. 下载必要的插件

    • markdown all in one
    • markdown preview enhanced
    • markdown shortcuts
    • markdownlint

  3. 创建.md文档，打开同步预览功能，开始编辑

二、基本语法#

  1. 标题

    #一级标题 ##二级标题 以此类推，最多支持六级标题即######

  2. 引用

    单个尖角号

    多个尖角号，多级引用

    a

    b

  3. 列表

    1. 无序列表
      • 列表一 -
      • 列表二 +
      • 列表三 *
    2. 有序列表
    3. 嵌套
    4. Todo列表
      • a
      • b
      • c

  4. 表格

    在表头下一行对应有几列就打几个|-|

    对齐方式 左对齐 居中对齐 右对齐
    a b c

  5. 换行

    • 换行？——两个以上空格后回车/空一行
    • 分割线——三个*
    • 字体
    字体 代码
    斜体 * *
    ==高亮== == ==
    粗体 ** **
    斜粗体 *** ***
    ~~~删除~~~ ~~~ ~~~
    下划线 <u> </u>
    • 脚注1

  6. 代码

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>cat</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

    <h1>Hello World!</h1>

  7. 超链接

    • 更多markdown教程可以参考网站:https://markdown.com.cn
    • 查看更多使用教程请点击链接 教程

  8. 图片

    • 使用图床保存图片插入 <img src="https://example.com/image" alt="text">

三、其他操作#

  • 插入latex公式
    • 行内显示公式 $f(x)=ax+b$
    • 块内显示函数教学 $$ \begin{Bmatrix} a & b \ c & d \end{Bmatrix} $$
    • html/css语法
      • ctrl+shift+p 搜索 "Markdown Preview Enhanced:Customize CSS" 在.markdown-preview.markdown-preview {}书写css样式
